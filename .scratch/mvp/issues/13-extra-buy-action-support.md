# 13. 추가 매수권 지원

Status: done

## Parent

.scratch/mvp/PRD.md

## What to build

자산 매수 후 특정 카드 효과 또는 규칙이 추가 매수권을 부여할 수 있게 하고, 추가 매수권 상태에서 같은 영업일에 시장 카드 또는 예약 카드 한 장을 더 매수할 수 있게 한다. 추가 매수권은 자산 매수 전용이며 예약과 자원 확보에는 사용할 수 없고, 중첩되거나 다음 영업일로 이월되지 않아야 한다.

## Code work

- ExtraBuyAction module을 구현한다.
- BusinessDayState에 추가 매수권 보유 여부, 추가 매수 선택 대기 여부, 추가 매수 중 구매 여부를 연결한다.
- 첫 번째 자산 매수 후 매수 관련 규칙이 GrantExtraBuyAction을 발생시키면 즉시 영업일을 종료하지 않고 추가 매수 선택 상태로 진입한다.
- 추가 매수권 상태에서 가능한 행동은 시장 카드 매수, 예약 카드 매수, 다음 영업일로 제한한다.
- 추가 매수권 상태에서는 예약과 자원 확보를 비활성화한다.
- 추가 매수권은 최대 1개이며 중첩되지 않는다.
- 추가 매수로 산 카드의 일반 매수 규칙은 발동하되 GrantExtraBuyAction만 무시한다.
- 추가 매수권을 사용해 두 번째 매수를 완료하면 추가 매수권을 소모하고 영업일을 종료한다.
- 다음 영업일을 누르면 추가 매수권을 포기하고 영업일을 종료한다.

## Unity Editor work

- 추가 매수권 상태 안내 UI를 만든다.
- 추가 매수권 상태에서는 중앙 은행을 비활성화한다.
- 시장 카드 상세보기에서는 예약 버튼을 숨기거나 비활성화한다.
- 다음 영업일 버튼은 추가 매수권 포기 버튼 역할을 하도록 현재 상태 안내를 표시한다.
- 테스트용 GrantExtraBuyAction 카드 또는 개발용 트리거를 데이터에 추가한다.

## Verification

- Play Mode에서 추가 매수권을 주는 카드를 매수하면 영업일이 즉시 종료되지 않고 추가 매수 가능 상태가 된다.
- 추가 매수권 상태에서 시장 카드 또는 예약 카드를 매수하면 영업일이 종료된다.
- 추가 매수권 상태에서 예약 버튼이 보이지 않거나 비활성화된다.
- 추가 매수권 상태에서 자원 확보를 할 수 없다.
- 추가 매수권 상태에서 다음 영업일을 누르면 권리가 사라지고 영업일이 종료된다.
- 추가 매수로 다시 추가 매수권을 얻으려 해도 중첩되지 않는다.

## Acceptance criteria

- [x] 추가 매수권은 자산 매수 후 발생할 수 있다.
- [x] 추가 매수권이 발생하면 첫 번째 매수 직후 영업일을 종료하지 않는다.
- [x] 추가 매수권으로 시장 카드와 예약 카드는 매수 가능하다.
- [x] 추가 매수권으로 예약과 자원 확보는 불가능하다.
- [x] 추가 매수권은 중첩되지 않는다.
- [x] 추가 매수권은 다음 영업일로 이월되지 않는다.
- [x] 추가 매수 후에는 즉시 영업일을 종료한다.
- [x] 추가 매수에서 발생한 GrantExtraBuyAction은 무시한다.
