# 16. 시장 자산 카드의 아이콘 중심 카드 문법

Status: done

## Parent

.scratch/mvp/UI_GAMEPLAY_SPEC.md

## What to build

시장 영역의 기본 상태를 UI_GAMEPLAY_SPEC의 시장 테이프 문법에 맞춘다. 매도 임박, 현재 시장, 예비 시장이 3열로 읽히고, 각 자산 카드는 현금 비용, 전문 자원 비용, 분기 운용 수익, 운용가치, 태그, 희귀도를 줄글이 아니라 배치와 아이콘 중심으로 보여준다. 예비 시장 카드는 현재 거래 대상이 아니라 미래 정보로 보이도록 작고 차분한 미리보기 카드로 구분한다.

이 slice는 최종 카드 프레임이나 카드 일러스트 완성이 아니라, 이후 애셋 교체가 가능한 카드 정보 배치와 시장 테이프 읽기 구조를 고정하는 것이 목표다.

## Acceptance criteria

- [x] 시장 영역은 매도 임박, 현재 시장, 예비 시장 3열을 유지한다.
- [x] 매도 임박과 현재 시장의 시장 카드는 클릭 가능한 매수/예약 대상처럼 보인다.
- [x] 예비 시장 카드는 더 작고 낮은 강조 상태로 보이며, 현재 매수/예약 대상처럼 보이지 않는다.
- [x] 자산 카드에는 카드 이름과 태그명 외의 설명문 줄글이 들어가지 않는다.
- [x] 현금 비용은 현금 아이콘 또는 지폐 배지와 숫자로 보인다.
- [x] 전문 자원 비용은 리서치, 신용, 원자재 아이콘 또는 칩 언어와 숫자로 보인다.
- [x] 분기 운용 수익과 운용가치는 서로 다른 위치와 시각적 무게로 구분된다.
- [x] 카드 정보는 placeholder 카드 아트가 있어도 비용, 수익, 운용가치, 태그를 가리지 않는다.
