## What to build

성격 변환형 Mr.Market 제안을 실제 매수와 보유 주식 상태에 연결한다. 성격 변환형은 구매 비용을 바꿀 수 있고, 매수 성공 시 가치/배당 영구 변화를 매입 이력으로 남긴다. 영구 변화는 오직 가치 변화와 배당 변화만 저장한다.

보유 주식의 최종 가치와 최종 배당은 원본 값과 영구 흔적의 합으로 계산하되 0 미만으로 내려가지 않는다. 호일 합성 시 재료 3장의 영구 흔적을 모두 합산해 호일에 승계한다. 매도 가격은 이 영구 흔적을 반영하지 않는다.

## Acceptance criteria

- [ ] 성격 변환형 제안은 매수 비용 변화를 결제 검증에 반영할 수 있다.
- [ ] 성격 변환형 제안은 매수 성공 시 가치 변화와 배당 변화만 영구 흔적으로 저장한다.
- [ ] 매수 실패는 영구 흔적을 저장하지 않는다.
- [ ] 보유 주식 최종 가치는 원본 가치와 영구 가치 흔적을 합산해 계산된다.
- [ ] 보유 주식 최종 배당은 원본 배당과 영구 배당 흔적을 합산해 계산된다.
- [ ] 최종 가치는 0 미만으로 내려가지 않는다.
- [ ] 최종 배당은 0 미만으로 내려가지 않는다.
- [ ] 미션 정산에서 최종 가치가 필요한 경우 영구 가치 흔적이 반영된다.
- [ ] 호일 합성은 재료 3장의 영구 가치 흔적을 모두 합산한다.
- [ ] 호일 합성은 재료 3장의 영구 배당 흔적을 모두 합산한다.
- [ ] 호일 배당 하한은 재료 흔적 합산 후 최종 적용된다.
- [ ] 호일 최종 가치와 최종 배당은 미션 정산에 사용할 수 있다.
- [ ] 주식 매도 가격은 Mr.Market 영구 가치/배당 흔적을 반영하지 않는다.
- [ ] 관련 EditMode 테스트가 성격 변환 매수, 하한 처리, 미션 계산 연결, 호일 흔적 합산, 매도 가격 제외를 검증한다.
- [ ] UI-facing 검증이 성격 변환 주식의 원본/최종 값 차이를 확인한다.
