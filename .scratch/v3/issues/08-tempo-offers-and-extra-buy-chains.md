## What to build

템포형 Mr.Market 제안을 실제 보상과 추가 매수권 상태에 연결한다. 템포형은 비용 변화와 영구 가치/배당 변화를 만들지 않는다. 대신 추가 매수권과 투자 철학 보너스를 지급할 수 있다.

추가 매수권은 같은 영업일에 시장 주식, 예약 주식, 소모형 자원 카드 중 하나를 추가로 살 수 있게 한다. 예약은 추가 매수권을 보존한다. 플레이어가 다음 영업일로 넘기면 사용하지 않은 추가 매수권은 사라진다. 템포 제안이 다시 추가 매수권을 주면 체인이 이어질 수 있으며, 시스템은 별도 체인 상한으로 강제 종료하지 않는다.

## Acceptance criteria

- [ ] 템포형 제안은 비용을 바꾸지 않는다.
- [ ] 템포형 제안은 가치/배당 영구 흔적을 만들지 않는다.
- [ ] 템포형 제안은 추가 매수권을 지급할 수 있다.
- [ ] 템포형 제안은 독서, 명상, 인내 보너스를 지급할 수 있다.
- [ ] 투자 철학 보너스는 기존 보유 한도를 넘는 초과분을 폐기한다.
- [ ] 추가 매수권으로 시장 주식을 구매할 수 있다.
- [ ] 추가 매수권으로 예약 주식을 구매할 수 있다.
- [ ] 추가 매수권으로 소모형 자원 카드를 구매할 수 있다.
- [ ] 추가 매수권 상태에서 주식을 구매하면 추가 매수권이 소모된다.
- [ ] 추가 매수권 상태에서 소모형 자원 카드를 구매하면 추가 매수권이 소모된다.
- [ ] 추가 매수권 상태에서 예약하면 추가 매수권이 유지된다.
- [ ] 다음 영업일 버튼을 누르면 사용하지 않은 추가 매수권이 폐기된다.
- [ ] 추가 매수권은 다음 영업일로 이월되지 않는다.
- [ ] 추가 매수로 산 주식은 다음 영업일 전까지 배당을 지급하지 않는다.
- [ ] 추가 매수로 템포 제안을 다시 소비하면 새 추가 매수권을 받을 수 있다.
- [ ] 같은 영업일에 여러 제안을 소비할 수 있다.
- [ ] 시장 땡김으로 새로 들어온 카드는 즉시 추가 매수 후보가 될 수 있다.
- [ ] 관련 EditMode 테스트가 보너스 지급, 추가 매수권 소비/보존/폐기, 체인 허용을 검증한다.
- [ ] PlayMode 또는 UI-facing 테스트가 추가 매수 상태의 버튼 가용성과 다음 영업일 폐기를 검증한다.
