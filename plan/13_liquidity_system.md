# 13. 소모형 자원 카드

## 1. 목적

이 문서는 중앙 은행을 대체하는 소모형 자원 카드의 종류, 비용, 효과, 한도 처리, 시장 테이프와의 관계, 재순환 규칙을 정의한다.

기존 중앙 은행 액션과 자원 확보 화면은 제거한다.

```text
중앙 은행 없음
자원 확보 화면 없음
GainLiquidity 상태 없음
```

플레이어는 시장에 등장한 소모형 자원 카드를 매수해 현금 또는 투자 철학을 획득한다.

---

## 2. 카드 성격

소모형 자원 카드는 포트폴리오에 등록되지 않는다.

```text
소모형 자원 카드 매수
→ 비용 지불
→ 자원 획득 효과 발동
→ 카드 제거
→ 시장 테이프 당김
→ 영업일 종료
```

소모형 자원 카드는 다음 값을 가지지 않는다.

```text
- 가치
- 배당금
- 호일 상태
- 포트폴리오 슬롯
```

---

## 3. 표시 정보

소모형 자원 카드 표시 정보:

```text
- 이미지
- 등급
- 제공 자원
- 코스트
```

이름은 표시하지 않는다.

등급은 카드 효율과 출현 빈도를 나타낸다.
최종 평가에는 직접 포함되지 않는다.

---

## 4. 카드 종류

### 4.1 현금 획득 카드

```text
현금 획득 카드
→ 비용 없음 또는 현금 비용만 가능
→ 매수 시 현금 N 획득
```

현금 획득 카드로 얻은 현금은 조달 현금이다.
현금 흐름에는 포함하지 않는다.

### 4.2 투자 철학 획득 카드

```text
투자 철학 획득 카드
→ 현금 비용만 요구
→ 매수 시 특정 투자 철학 N 획득
```

투자 철학:

```text
- 독서
- 명상
- 인내
```

소모형 자원 카드에는 투자 철학 비용 슬롯이 없다.
딜도 사용할 수 없다.

---

## 5. 투자 철학 한도

투자 철학은 합산 한도와 개별 한도를 모두 가진다.

```text
독서 + 명상 + 인내 <= 10
독서 <= 5
명상 <= 5
인내 <= 5
```

현금과 딜은 투자 철학 한도에 포함하지 않는다.

투자 철학 획득 시 한도를 초과하면 신규 획득분 중 초과분만 폐기한다.

예시:

```text
현재:
독서 5
명상 3
인내 1
합계 9 / 10

독서 +2 획득 시도
→ 독서 개별 한도 초과
→ 독서 +2 폐기
```

다른 예시:

```text
현재:
독서 4
명상 3
인내 2
합계 9 / 10

독서 +2 획득 시도
→ 독서 +1 획득
→ 독서 +1 폐기
→ 합계 10 / 10
```

메시지:

```text
투자 철학 최대 보유: 독서 +1 폐기
```

---

## 6. 매수 가능 조건

소모형 자원 카드 매수 조건:

```text
- 선택 카드가 소모형 자원 카드이다.
- 최종 현금 비용을 지불할 수 있다.
- 현재 매수 가능한 행동 상태이다.
```

소모형 자원 카드 결제에는 다음을 사용하지 않는다.

```text
- 투자 철학 칩
- 딜
```

딜은 투자 철학 마스터리로 전환하는 보상 자원이며, 어떤 카드 결제에도 직접 사용하지 않는다.

---

## 7. 시장 테이프 처리

소모형 자원 카드는 시장 8칸에 주식 카드와 함께 등장한다.

매수 후 처리:

```text
소모형 자원 카드 매수
→ 효과 발동
→ 시장 슬롯 비움
→ 시장 테이프 당김
```

소모형 자원 카드는 예약할 수 없다.

```text
소모형 자원 카드
→ 예약 불가
```

---

## 8. 덱과 재순환

소모형 자원 카드는 소모형 자원 덱에서 뽑는다.

시장 새 카드 뽑기에서 소모형 자원 덱은 25% 확률로 선택된다.

```text
새 카드 뽑기:
75% 주식 덱
25% 소모형 자원 덱
```

소모형 자원 카드는 매수 후 재순환 가능한 풀로 이동할 수 있다.
소모형 자원 덱이 고갈되면 재순환 가능한 풀을 되돌리고 셔플한다.

```text
소모형 자원 덱 고갈
→ 재순환 풀을 덱에 되돌림
→ 셔플
→ 다시 뽑기
```

소모형 자원 카드가 덱/풀에 들어간 경우 해당 덱/풀을 한 번 셔플한다.

---

## 9. 수익 기록

소모형 자원 카드로 얻은 현금은 조달 현금이다.

```text
소모형 현금 카드
→ CurrentCash 증가
→ CurrentQuarterRevenue 증가 없음
→ TotalRevenue 증가 없음
```

투자 철학 획득도 수익에 포함하지 않는다.

분기 목표 판정에 포함되는 수익은 다음이다.

```text
- 배당금
- 주식 매도 수익
```

v3 기준 월세 밀림 판정에는 위 현금 흐름에 미션 수익을 더한다.
미션 수익은 현금이 아니므로 소모형 자원 카드 수익 기록에는 포함하지 않는다.

## 11. 구현 시 주의사항

```text
- 중앙 은행 액션은 제거한다.
- 자원 확보 화면은 제거한다.
- 시장 영역은 자원 확보 상태로 전환되지 않는다.
- 소모형 자원 카드는 시장에 등장하는 카드이다.
- 소모형 자원 카드는 매수만 가능하고 예약 불가다.
- 소모형 자원 카드는 포트폴리오에 등록되지 않는다.
- 현금 획득 카드의 현금은 수익에 포함하지 않는다.
- 투자 철학 획득 카드의 비용은 현금만 사용한다.
- 소모형 자원 카드에는 투자 철학 비용 슬롯이 없다.
- 소모형 자원 카드 결제에는 딜을 사용할 수 없다.
- 매수 후 시장 빈칸은 시장 테이프 당김으로 처리한다.
- 소모형 자원 카드는 재순환 가능하다.
```
