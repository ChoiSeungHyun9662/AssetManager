# 05. 자원 구조

## 1. 목적

이 문서는 새 주식 규칙에서 사용하는 자원의 종류, 역할, 보유 한도, 획득/소비 규칙을 정의한다.

자원은 다음으로 구성한다.

```text
- 현금
- 투자 철학
  - 독서
  - 명상
  - 인내
- 딜
```

---

## 2. 현금

현금은 주식과 소모형 자원 카드를 매수할 때 사용하는 기본 결제 자원이다.

현금은 증가 원인에 따라 기록 방식이 다르다.

```text
조달 현금
→ 소모형 현금 카드 등으로 얻은 현금
→ 현금 흐름에 포함하지 않음

현금 흐름
→ 배당금, 주식 매도 현금
→ 분기 평가 목표 판정에 포함
```

현금 보유 한도는 없다.

---

## 3. 투자 철학

투자 철학은 주식 매수 시 비용 슬롯에 배치하는 칩형 자원이다.

투자 철학은 다음 3종이다.

```text
- 독서
- 명상
- 인내
```

각 투자 철학은 해당 비용 슬롯에만 배치할 수 있다.

```text
독서 칩 → 독서 슬롯
명상 칩 → 명상 슬롯
인내 칩 → 인내 슬롯
```

---

## 4. 투자 철학 보유 한도

투자 철학은 합산 한도와 개별 한도를 모두 가진다.

```text
독서 + 명상 + 인내 <= 10
독서 <= 5
명상 <= 5
인내 <= 5
```

현금과 딜은 투자 철학 한도에 포함하지 않는다.

예시:

```text
독서 5
명상 3
인내 2
→ 합계 10 / 10
→ 투자 철학 추가 획득 불가
```

```text
독서 5
명상 0
인내 0
→ 독서는 개별 한도 도달
→ 명상/인내는 합산 한도 범위 안에서 획득 가능
```

---

## 5. 투자 철학 초과 처리

투자 철학 획득 시 한도를 넘으면 기존 보유 자원은 건드리지 않는다.
신규 획득분 중 초과분만 폐기한다.

초과 판정은 개별 한도와 합산 한도를 모두 적용한다.

예시:

```text
현재:
독서 5
명상 3
인내 1
합계 9 / 10

독서 +2 획득 시도
→ 독서 개별 한도 초과
→ 독서 +2 폐기
```

```text
현재:
독서 4
명상 3
인내 2
합계 9 / 10

독서 +2 획득 시도
→ 독서 +1 획득
→ 독서 +1 폐기
→ 최종 합계 10 / 10
```

메시지:

```text
투자 철학 최대 보유: 독서 +1 폐기
투자 철학 최대 보유: 명상 +2 폐기
```

여러 자원이 동시에 폐기되면 한 메시지에 묶어서 표시할 수 있다.

---

## 6. 딜

딜은 포트폴리오 성장으로 얻는 특수 보상 자원이다.

```text
- 보유 주식 슬롯 3칸 최초 도달 → 딜 +1
- 보유 주식 슬롯 5칸 최초 도달 → 딜 +1
- 보유 주식 슬롯 8칸 최초 도달 → 딜 +1
- 첫 호일 주식 완성 → 딜 +1
```

각 조건은 한 런에서 한 번만 보상한다.
딜은 투자 철학 마스터리로 전환하기 전까지 보유한다.

```text
딜 보유 한도 없음
```

딜은 현금이나 투자 철학 보유 한도에 포함하지 않는다.

---

## 7. 딜과 투자 철학 마스터리

딜은 주식 매수 결제에 직접 사용하지 않는다.
딜 1개를 독서/명상/인내 중 하나에 전환해 해당 투자 철학 마스터리를 1 올린다.

```text
딜 1개
→ 독서/명상/인내 중 1종 선택
→ 해당 투자 철학 마스터리 +1
```

마스터리 한도:

```text
각 투자 철학 마스터리 <= 5
```

마스터리는 주식 매수의 같은 종류 투자 철학 비용을 낮춘다.

```text
최종 투자 철학 비용
= max(0, 원래 투자 철학 비용 - 해당 마스터리)
```

딜은 투자 철학 비용 슬롯에 배치하지 않는다.
딜은 현금 비용을 낮추지 않는다.

---

## 8. 자원 획득 경로

현금 획득:

```text
- 소모형 현금 카드 매수
- 보유 주식의 배당금
- 주식 매도 수익
```

투자 철학 획득:

```text
- 투자 철학 획득 소모형 자원 카드 매수
```

딜 획득:

```text
- 보유 주식 슬롯 3/5/8 최초 도달
- 첫 호일 주식 완성
```

기존 중앙 은행 자원 확보 액션은 폐기한다.

---

## 9. 수익 기록

수익에 포함하는 현금:

```text
- 배당금
- 주식 매도 수익
- 기타 수익으로 명시된 현금
```

미션 수익은 현금이 아니므로 `AddRevenueCash`로 누적하지 않는다.
분기 목표 판정에서는 현금 흐름에 미션 수익을 더해 별도로 계산한다.

수익에서 제외하는 현금:

```text
- 소모형 현금 카드로 얻은 현금
```

추천 함수 구분:

```csharp
void AddFundingCash(int amount)
{
    CurrentCash += amount;
}
```

```csharp
void AddRevenueCash(int amount)
{
    CurrentCash += amount;
    CurrentQuarterRevenue += amount;
    CurrentFiscalYearRevenue += amount;
    TotalRevenue += amount;
}
```

## 11. 구현 시 주의사항

```text
- 전문 자원이라는 용어를 사용하지 않는다.
- 리서치/신용/원자재 대신 독서/명상/인내를 사용한다.
- 투자 철학은 합산 10개, 개별 5개 한도를 가진다.
- 현금과 딜은 투자 철학 한도에 포함하지 않는다.
- 투자 철학 초과분은 신규 획득분에서만 폐기한다.
- 딜 보유 한도는 없다.
- 딜은 주식 매수 결제에 직접 사용하지 않는다.
- 딜은 투자 철학 마스터리로 전환한다.
- 각 투자 철학 마스터리는 최대 5이다.
- 마스터리는 같은 종류 투자 철학 비용을 낮춘다.
- 소모형 현금 카드로 얻은 현금은 수익에 포함하지 않는다.
- 배당금과 주식 매도 수익은 수익에 포함한다.
```
