# 10. 칩 조작 UX

## 1. 목적

이 문서는 주식 매수 과정에서 사용하는 칩 조작 UX를 정의한다.

칩으로 표현하는 대상:

```text
- 독서
- 명상
- 인내
```

현금은 칩으로 배치하지 않는다.
현금은 매수 확정 시 자동 차감한다.

---

## 2. 칩 종류

### 2.1 투자 철학 칩

```text
독서 칩 → 독서 슬롯
명상 칩 → 명상 슬롯
인내 칩 → 인내 슬롯
```

투자 철학 칩은 같은 종류의 비용 슬롯에만 배치할 수 있다.

### 2.2 투자 철학 마스터리

딜은 비용 슬롯에 배치하는 칩이 아니다.
딜을 전환해 얻은 투자 철학 마스터리는 같은 종류의 투자 철학 비용을 자동으로 낮춘다.

```text
최종 독서 비용 = max(0, 원래 독서 비용 - 독서 마스터리)
```

---

## 3. 칩 배치 대상

칩 배치는 주식 매수 비용 슬롯에서만 발생한다.

소모형 자원 카드에는 투자 철학 비용 슬롯이 없다.
소모형 자원 카드 결제에는 딜을 사용할 수 없다.

---

## 4. 비용 슬롯 상태

각 비용 슬롯은 다음 상태 중 하나를 가진다.

```text
- 비어 있음
- 해당 투자 철학 칩 배치됨
```

투자 철학 마스터리로 줄어든 슬롯은 아예 요구 수량이 감소한 것으로 본다.
칩이 올라간 슬롯은 결제 완료 상태로 본다.

---

## 5. 배치 방식

MVP에서 지원할 수 있는 방식:

```text
- 클릭 배치
- 칩 회수
- 자동 배치
```

가능하면 지원:

```text
- 드래그 배치
- 배치 가능 슬롯 하이라이트
```

---

## 6. 유효 배치

유효 배치 조건:

```text
- 현재 주식 매수 결제 중이다.
- 대상 슬롯이 비어 있다.
- 대상 슬롯이 해당 칩을 받을 수 있다.
- 플레이어가 해당 칩을 보유하고 있다.
```

예시:

```text
독서 칩 → 빈 독서 슬롯 = 가능
명상 칩 → 빈 독서 슬롯 = 불가
```

---

## 7. 소비 타이밍

칩을 슬롯에 올려도 자원은 아직 소비되지 않는다.

```text
칩 배치
→ 소비 아님
```

자원은 매수 확정 시점에만 소비한다.

```text
매수 확정
→ 배치된 투자 철학 칩 소비
→ 현금 차감
```

매수 확정 전에는 언제든 칩을 회수할 수 있다.

---

## 8. 자동 배치

자동 배치는 가능한 칩을 비용 슬롯에 자동으로 배치한다.

기본 우선순위:

```text
1. 투자 철학 마스터리 적용 후 남은 요구 슬롯 계산
2. 해당 투자 철학 칩으로 채울 수 있는 슬롯 배치
3. 최종 현금 비용 표시
```

자동 배치는 매수를 자동 확정하지 않는다.

---

## 9. 최종 비용 계산

투자 철학 마스터리는 투자 철학 비용에만 반영한다.
현금 비용은 카드 비용과 Mr.Market 제안 같은 비용 수정 효과를 먼저 반영한 뒤 인플레이션을 적용한다.

```text
기본 현금 비용
→ 비용 수정 효과
→ 인플레이션 적용
→ 최종 현금 비용
```

예시:

```text
기본 현금 비용 5
Mr.Market 현금 비용 -1
인플레이션 +1
→ 최종 현금 비용 5
```

## 11. 구현 시 주의사항

```text
- 칩 조작은 독서/명상/인내에만 적용한다.
- 현금은 칩으로 배치하지 않는다.
- 소모형 자원 카드에는 칩을 배치하지 않는다.
- 딜은 비용 슬롯에 배치하지 않는다.
- 딜은 투자 철학 마스터리로 전환한다.
- 투자 철학 마스터리는 같은 종류 투자 철학 비용을 자동으로 낮춘다.
- 칩 배치만으로 자원을 소비하지 않는다.
- 자원 소비는 매수 확정 시점에만 발생한다.
- 자동 배치는 매수를 자동 확정하지 않는다.
```
