# 10. 칩 조작 UX

## 1. 목적

이 문서는 주식 매수 과정에서 사용하는 칩 조작 UX를 정의한다.

칩으로 표현하는 대상:

```text
- 독서
- 명상
- 인내
- 딜
```

현금은 칩으로 배치하지 않는다.
현금은 매수 확정 시 자동 차감한다.

---

## 2. 칩 종류

### 2.1 투자 철학 칩

```text
독서 칩 → 독서 슬롯
명상 칩 → 명상 슬롯
인내 칩 → 인내 슬롯
```

투자 철학 칩은 같은 종류의 비용 슬롯에만 배치할 수 있다.

### 2.2 딜

딜은 모든 투자 철학 비용 슬롯에 배치할 수 있다.

```text
딜 → 독서 슬롯 가능
딜 → 명상 슬롯 가능
딜 → 인내 슬롯 가능
```

딜 1개:

```text
투자 철학 슬롯 1칸 대체
+ 기본 현금 비용 1 감소
```

---

## 3. 칩 배치 대상

칩 배치는 주식 매수 비용 슬롯에서만 발생한다.

소모형 자원 카드에는 투자 철학 비용 슬롯이 없다.
소모형 자원 카드 결제에는 딜을 사용할 수 없다.

---

## 4. 비용 슬롯 상태

각 비용 슬롯은 다음 상태 중 하나를 가진다.

```text
- 비어 있음
- 해당 투자 철학 칩 배치됨
- 딜 배치됨
```

칩 또는 딜이 올라간 슬롯은 결제 완료 상태로 본다.

---

## 5. 배치 방식

MVP에서 지원할 수 있는 방식:

```text
- 클릭 배치
- 칩 회수
- 자동 배치
```

가능하면 지원:

```text
- 드래그 배치
- 배치 가능 슬롯 하이라이트
```

---

## 6. 유효 배치

유효 배치 조건:

```text
- 현재 주식 매수 결제 중이다.
- 대상 슬롯이 비어 있다.
- 대상 슬롯이 해당 칩을 받을 수 있다.
- 플레이어가 해당 칩을 보유하고 있다.
```

예시:

```text
독서 칩 → 빈 독서 슬롯 = 가능
딜 → 빈 인내 슬롯 = 가능
명상 칩 → 빈 독서 슬롯 = 불가
```

---

## 7. 소비 타이밍

칩을 슬롯에 올려도 자원은 아직 소비되지 않는다.

```text
칩 배치
→ 소비 아님
```

자원은 매수 확정 시점에만 소비한다.

```text
매수 확정
→ 배치된 투자 철학 칩 소비
→ 배치된 딜 소비
→ 현금 차감
```

매수 확정 전에는 언제든 칩을 회수할 수 있다.

---

## 8. 자동 배치

자동 배치는 가능한 칩을 비용 슬롯에 자동으로 배치한다.

기본 우선순위:

```text
1. 해당 투자 철학 칩으로 채울 수 있는 슬롯 우선
2. 부족한 슬롯은 딜로 보완
3. 딜 할인 적용 후 현금 비용 재계산
```

자동 배치는 매수를 자동 확정하지 않는다.

---

## 9. 딜 할인 계산

딜 할인은 인플레이션 적용 전에 반영한다.

```text
기본 현금 비용
→ 딜 할인
→ 인플레이션 적용
→ 최종 현금 비용
```

예시:

```text
기본 현금 비용 5
딜 2개 사용
인플레이션 +1
→ 최종 현금 비용 4
```

## 11. 구현 시 주의사항

```text
- 칩 조작은 독서/명상/인내/딜에만 적용한다.
- 현금은 칩으로 배치하지 않는다.
- 소모형 자원 카드에는 칩을 배치하지 않는다.
- 딜은 모든 투자 철학 슬롯에 배치 가능하다.
- 딜은 슬롯 대체와 현금 비용 감소를 동시에 수행한다.
- 딜 할인은 인플레이션 적용 전에 반영한다.
- 칩 배치만으로 자원을 소비하지 않는다.
- 자원 소비는 매수 확정 시점에만 발생한다.
- 자동 배치는 매수를 자동 확정하지 않는다.
```
