# 22. 주식 규칙 대개편

## 1. 목적

이 문서는 주식 중심 규칙 개편의 기준 문서이다.

기존 세부 문서에 남아 있는 `자산`, `전문 자원`, `중앙 은행`, `예약 구역`, `환매 압력`, `카드 상세보기` 규칙과 충돌하는 경우 이 문서를 우선한다.

본 개편의 목표는 게임 정체성을 다음 방향으로 수렴시키는 것이다.

```text
주식만 포트폴리오에 담는 카드 엔진 빌딩 게임
시장 8칸에서 주식과 소모형 자원 카드를 사고, 좋은 주식은 예약으로 잠그는 게임
배당금, 매도 현금, 미션 수익으로 분기 평가 목표를 넘기고, 월세 밀림을 관리하는 게임
```

---

## 2. 용어 변경

| 기존 용어 | 새 용어 |
|---|---|
| 자산 | 주식 |
| 보유 자산 | 보유 주식 |
| 운용가치 | 가치 |
| 운용 수익 | 배당금 또는 수익 |
| 전문 자원 | 투자 철학 |
| 리서치 | 독서 |
| 신용 | 명상 |
| 원자재 | 인내 |
| 환매 압력 | 월세 밀림 |
| 런 실패 | 파산 |

`배당금`과 `수익`은 구분한다.

```text
배당금
→ 보유 주식이 영업일 시작에 제공하는 현금

수익
→ 현금 흐름 성과를 보여주는 누적값
→ 배당금 + 주식 매도 현금
```

소모형 자원 카드로 얻는 현금은 조달 현금이며 수익에 포함하지 않는다.
v3 분기 평가 목표 판정은 현금 흐름에 미션 수익을 더해 계산한다.

## 3. 자원 구조

자원은 다음으로 구성한다.

```text
- 현금
- 투자 철학
  - 독서
  - 명상
  - 인내
- 딜
```

투자 철학은 칩 형태를 유지한다.

```text
독서 + 명상 + 인내 <= 10
독서 <= 5
명상 <= 5
인내 <= 5
```

현금과 딜은 투자 철학 한도에 포함하지 않는다.

투자 철학 획득 시 합산 한도 또는 개별 한도를 넘는 초과분은 신규 획득분에서만 폐기한다.

```text
초과 폐기 메시지:
투자 철학 최대 보유: [자원명] +[수량] 폐기
```

딜은 투자 철학 마스터리로 전환하는 보상 자원으로 유지한다.

```text
딜 1개
→ 독서/명상/인내 중 1종 선택
→ 해당 투자 철학 마스터리 +1

각 투자 철학 마스터리 <= 5
주식 매수 시 같은 종류 투자 철학 비용을 마스터리만큼 감소
```

딜 자체는 주식 매수 결제에 직접 사용하지 않는다.
딜은 투자 철학 비용 슬롯을 대체하지 않고, 현금 비용도 낮추지 않는다.

## 4. 카드 종류

시장에 등장하는 카드는 두 종류이다.

```text
1. 주식 카드
2. 소모형 자원 카드
```

### 4.1 주식 카드

주식 카드는 포트폴리오에 등록되는 카드이다.

표시 정보:

```text
- 이미지
- 이름
- 등급
- 가치
- 코스트
- 배당금
```

주식 데이터는 기본 상태와 호일 상태 수치를 함께 가진다.

```text
- 이름
- 이미지
- 등급
- 기본 가치
- 기본 배당금
- 호일 가치
- 호일 배당금
- 코스트
```

호일 상태의 등급은 기본 상태와 같다.
호일 가치와 호일 배당금은 기획자가 직접 지정하며, 배율로 자동 계산하지 않는다.

### 4.2 소모형 자원 카드

소모형 자원 카드는 매수 시 효과를 발동하고 사라지는 카드이다.
포트폴리오에 등록되지 않으며, 가치/배당금/호일 상태를 가지지 않는다.

표시 정보:

```text
- 이미지
- 등급
- 제공 자원
- 코스트
```

소모형 자원 카드의 등급은 효율과 출현 빈도를 나타낸다.
최종 평가에는 직접 포함되지 않는다.

소모형 자원 카드 종류:

```text
현금 획득 카드
→ 비용 없음 또는 현금 비용만 가능
→ 매수 시 현금 N 획득

투자 철학 획득 카드
→ 현금 비용만 요구
→ 매수 시 특정 투자 철학 N 획득
```

## 5. 시장 테이프

시장 테이프는 1x8 구조이다.
기존 `매도 임박 / 현재 / 예비` 구역 구분은 제거한다.

```text
시장 슬롯: 왼쪽부터 1~8번
1번 슬롯: 가장 먼저 사라지는 위치
8번 슬롯: 새 카드가 등장하는 위치
```

카드는 오른쪽에서 등장해 왼쪽으로 이동하고 사라진다.

### 5.1 시장 테이프 진행

시장 테이프 진행은 새 영업일 시작 시 발생한다.
단, 분기 첫 영업일에는 시장 테이프 갱신만 하고 진행은 생략한다.

예약된 주식이 없다면 다음처럼 처리한다.

```text
1번 카드 제거
2번 → 1번
3번 → 2번
...
8번 → 7번
8번에 새 카드 등장
```

예약된 주식이 있다면 제거 대상은 가장 왼쪽의 비예약 카드이다.

```text
시장 테이프 진행:
1. 가장 왼쪽의 비예약 카드 1장을 제거한다.
2. 예약된 주식은 위치를 고정한다.
3. 남은 비예약 카드를 왼쪽으로 한 칸 진행시킨다.
4. 가장 오른쪽의 비예약 빈칸에 새 카드 1장이 등장한다.
```

### 5.2 시장 테이프 갱신

시장 테이프 갱신은 각 분기 시작 시 발생한다.

```text
분기 첫 영업일 시작
→ 시장 테이프 갱신
→ 시장 테이프 진행 없음
```

갱신 시 예약된 주식 슬롯은 유지한다.
예약되지 않은 시장 슬롯만 제거 후 새 카드로 채운다.

### 5.3 시장 테이프 당김

시장에 빈칸이 생기면 시장 테이프 당김을 적용한다.

```text
빈칸 기준 오른쪽 카드들만 왼쪽으로 당긴다.
예약된 주식은 움직이지 않는다.
오른쪽 카드가 왼쪽으로 이동할 때 예약된 주식 슬롯은 건너뛴다.
가장 오른쪽의 비예약 빈칸에 새 카드가 등장한다.
```

복수의 빈칸이 생기면 가장 왼쪽 빈칸부터 순서대로 당김을 각각 적용한다.

시장 카드 매수, 예약된 주식 매수, 소모형 자원 카드 매수, 호일 완성에 따른 시장 카드 제거는 모두 시장 테이프 당김으로 빈칸을 메운다.

## 6. 카드 공급 덱

시장 카드 공급원은 두 덱으로 나눈다.

```text
- 주식 덱
- 소모형 자원 덱
```

새 카드 1장을 뽑을 때마다 다음 확률로 대상 덱을 고른다.

```text
75% → 주식 덱
25% → 소모형 자원 덱
```

이 비율은 시장 8칸을 항상 주식 6장/소모형 2장으로 맞추는 하드 규칙이 아니다.
새 카드를 뽑을 때마다 적용하는 소프트 규칙이다.

선택한 덱이 비어 있으면 반대 덱에서 뽑는다.

```text
1. 75%/25% 확률로 대상 덱 선택
2. 선택한 덱에서 뽑을 수 있으면 1장 뽑음
3. 선택한 덱이 비었으면 반대 덱에서 뽑음
4. 소모형 자원 덱이 비었고 재순환 풀이 있으면 되돌리고 셔플한 뒤 뽑음
5. 두 덱 모두 뽑을 수 없으면 예외 상태 표시
```

주식 덱이 고갈되면 주식은 더 이상 새로 등장하지 않는다.
이 경우 주식 덱이 선택될 때 소모형 자원 덱으로 fallback한다.

소모형 자원 카드는 매수 후 재순환 가능한 풀로 이동할 수 있다.
소모형 자원 카드가 덱/풀에 다시 들어가면 해당 덱/풀을 셔플한다.

제거된 주식, 매도된 주식, 호일 완성으로 제거된 주식은 덱에 되돌리지 않는다.

MVP에서는 충분한 카드 풀을 전제로 하여 두 덱 모두 뽑을 수 없는 예외가 발생하지 않게 설계한다.

## 7. 예약

예약은 시장 8칸 안의 특정 주식을 시장 테이프 진행/당김/갱신의 영향을 받지 않도록 잠그는 액션이다.
예약 구역은 제거한다.
별도 `Reservation Panel`이나 예약 카드 목록 UI는 두지 않는다.
예약 상태와 예약 주식 매수는 해당 시장 슬롯 카드에서 직접 확인하고 처리한다.

```text
동시 예약 가능 주식: 최대 1장
예약된 주식은 시장 8칸 안에 그대로 남음
예약된 주식은 시장 테이프 진행/당김/갱신의 영향을 받지 않음
```

예약 가능 대상:

```text
주식 카드만 가능
소모형 자원 카드 예약 불가
이미 예약된 주식 재예약 불가
```

예약 액션 처리:

```text
주식 예약
→ 선택 주식 예약 상태로 잠금
→ 영업일 소비 없음
→ 딜 획득 없음
→ 월세 밀림 증가 없음
```

예약 액션 직후에는 시장 테이프를 진행하지 않는다.
예약은 영업일을 소비하지 않으므로 플레이어는 같은 영업일에 다른 행동을 계속할 수 있다.
시장은 다음 영업일 시작 시 진행된다.
예약은 카드만 고정하며 Mr.Market 제안은 고정하지 않는다.

예약된 주식을 매수하면 예약 상태가 해제되고 해당 슬롯이 비며, 빈칸은 시장 테이프 당김으로 처리한다.

## 8. 포트폴리오와 호일

포트폴리오는 최대 8개의 주식만 보유할 수 있다.

```text
포트폴리오 최대 점유: 8칸
동일 주식 2장 보유 시 2칸 점유
동일 주식 3장 보유 시 즉시 1장 호일 주식으로 합침
```

포트폴리오 8칸이 모두 점유된 상태에서 신규 주식 매수를 시도하면 실패한다.

```text
포트폴리오 8/8
동일 주식 2장 보유 상황이 아님
→ 매수 불가
→ "주식 매도가 필요합니다" 알림
→ 비용 소비 없음
→ 영업일 소비 없음
```

예외:

```text
포트폴리오 8/8
동일 주식 2장을 이미 보유
시장의 3번째 동일 주식 매수 시도
→ 비용 지불 가능하면 매수 허용
→ 즉시 3장을 1장 호일 주식으로 합침
→ 포트폴리오 점유: 8칸 → 7칸
→ 영업일 종료
```

호일 완성 처리:

```text
동일 주식 3장 보유 달성
→ 가장 먼저 보유한 동일 주식 칸이 호일 주식으로 변함
→ 나머지 동일 주식 2장은 제거
→ 포트폴리오 빈칸 2개는 그대로 빈칸으로 남김
→ 이후 새 주식은 가장 왼쪽 빈 포트폴리오 칸에 들어감
```

호일 완성 시 제거 대상:

```text
- 덱에 남은 같은 종목 주식
- 시장에 있는 같은 종목 주식
- 예약 상태로 시장에 잠긴 같은 종목 주식
```

예약된 같은 종목 주식이 제거되면 예약 수가 감소한다.
시장 빈칸은 시장 테이프 당김으로 처리한다.

각 주식 종목은 덱에 여러 장 포함될 수 있다.
종목별 포함 장수는 데이터 테이블의 min~max 값으로 컨트롤한다.

## 9. 주식 매수와 소모형 카드 매수

영업일을 지나게 하는 액션:

```text
- 주식 매수
- 소모형 자원 카드 매수
- 다음 영업일
```

영업일을 지나게 하지 않는 액션:

```text
- 주식 예약
- 주식 매도
```

주식 매수:

```text
결제 검증
→ 투자 철학 마스터리 할인 반영
→ 투자 철학 소비
→ 현금 차감
→ 포트폴리오 추가 또는 호일 처리
→ 시장 슬롯 비움
→ 시장 테이프 당김
→ 추가 매수권 처리 또는 영업일 종료
```

소모형 자원 카드 매수:

```text
결제 검증
→ 현금 차감
→ 자원 획득 효과 발동
→ 카드 제거
→ 시장 테이프 당김
→ 영업일 종료
```

소모형 자원 카드는 투자 철학 비용을 사용하지 않는다.
딜은 어떤 카드 결제에도 직접 사용하지 않는다.

## 10. 주식 매도

주식 매도는 영업일을 소비하지 않으며 하루 횟수 제한이 없다.
매도 입력은 포트폴리오의 `Owned Stock Card 1~8`에서 제공한다.
보유 주식 카드의 Card Button에 마우스를 올리면 해당 카드의 Sell Button을 표시한다.
Card Button과 Sell Button 사이로 커서를 이동하는 동안에는 Sell Button을 계속 표시한다.
두 영역을 모두 벗어나면 Sell Button을 숨긴다.
Card Button 클릭은 Sell Button 노출 조건으로 사용하지 않는다.

매도 보상:

```text
일반 주식 매도
→ 현금 1 * 인플레이션 획득

호일 주식 매도
→ 현금 3 * 인플레이션 획득
```

주식 종류와 무관하게 보상이 같다.

매도 수익은 분기 목표 판정에 사용하는 수익에 포함한다.

매도한 주식은 이번 게임에서 제거된다.
덱/시장에 다시 등장하지 않는다.

## 11. 카드 표시 방식

기존 카드 상세보기 화면은 제거한다.

```text
시장 영역 상태는 Market 하나만 유지한다.
카드를 클릭해 별도 상세보기 상태로 전환하지 않는다.
카드에 마우스를 호버하면 크게 확대된 버전이 뜬다.
시장 카드와 호버 확대 카드가 제공하는 정보는 동일하며 스케일만 다르다.
```

중앙 은행 액션과 자원 확보 화면도 제거한다.
자원 획득은 시장에 등장하는 소모형 자원 카드를 매수해서 처리한다.

## 12. 월세 밀림과 파산

`월세 밀림`은 기존 환매 압력의 새 이름이다.

```text
월세 밀림 한도: 10
월세 밀림 >= 10
→ 파산
→ 게임 오버
```

월세 밀림 증가 요인:

```text
- 분기 평가 목표 미달 시 달성률에 따라 +1/+2/+3
```

분기 목표 미달 증가량:

```text
수익 >= 목표
→ +0

목표의 75% 이상 100% 미만
→ +1

목표의 50% 이상 75% 미만
→ +2

목표의 50% 미만
→ +3
```

월세 밀림은 증가 즉시 한도 검사를 한다.
10 이상이 되면 이후 영업일/분기/회계년도 진행을 중단하고 파산 화면으로 이동한다.

## 13. 분기와 영업일 타이밍

일반 영업일 시작:

```text
시장 테이프 진행
→ 보유 주식 배당금 지급
→ 플레이어 입력 대기
```

분기 첫 영업일 시작:

```text
시장 테이프 갱신
→ 시장 테이프 진행 없음
→ 보유 주식 배당금 지급
→ 플레이어 입력 대기
```

마지막 영업일 종료 후에는 다음 영업일이 없으므로 배당금이 발생하지 않고 분기 마감으로 이동한다.

분기 마감:

```text
확정 미션이 있으면 미션 수익 계산
→ 현금 흐름과 미션 수익 확정
→ 목표 달성률 계산
→ 월세 밀림 증가량 계산
→ 월세 밀림 한도 검사
→ 파산하지 않았다면 다음 일정 진행
```

## 15. MVP 반영 메모

MVP에서 반드시 반영해야 하는 변경:

```text
- 주식 전용 포트폴리오 8칸
- 시장 1x8
- 시장 테이프 진행/갱신/당김
- 예약 구역 제거 및 시장 슬롯 예약
- 딜과 투자 철학 마스터리 유지
- 월세 밀림 10 도달 시 파산
- 중앙 은행 제거
- 소모형 자원 카드
- 주식/소모형 자원 카드 별도 덱
- 호버 확대 카드
- 호일 합성
- 주식 매도
- 보유 주식 카드 호버 기반 매도 버튼
```

기존 문서에서 후속으로 제거해야 하는 표현:

```text
- 자산
- 전문 자원
- 리서치 / 신용 / 원자재
- 운용가치 / 운용 수익
- 환매 압력 / 대규모 환매 / 런 실패
- 카드 상세보기 상태
- 자원 확보 상태
- 중앙 은행
- 예약 구역
- 매도 임박 / 현재 시장 / 예비 시장
```
